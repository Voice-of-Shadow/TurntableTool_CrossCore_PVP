# 转盘工具_交错战线_PVP

### 说明

为群友组织的交错战线第一届PVP开发的一个静态转盘抽签页面。
因为没有找到比较合适的抽签工具。所以自己整了一个。
加了点对背景图像的处理。
可以替换项目中的资源内容和文本用于其他对战的抽签形式。
本项目图像和音频资源归属交错官方米茄所有。

---

### 快速开始

1. **启动工具**
    双击运行 `index.html` 文件，进入工具界面。点击任意网页位置直接激活BGM播放。
    ✅ 建议按 **F11** 进入全屏模式，以获得最佳显示效果。
2. **导入选手数据**
    将准备好的 `选手数据.txt` 文件拖入页面。
    ⚠️ 注意：工具将自动截取每行前 16 个字符作为选手名称，并载入最多 32 行数据。
3. **开始抽取**
    点击转盘进行抽选：

    - 奇数次抽取 → 红方选手
    - 偶数次抽取 → 蓝方选手
    - 红蓝方选手将自动配对为对战组合。
4. **完成与导出**
    全部选手抽取完毕后，工具将自动生成并下载 `对战列表.txt` 文件至浏览器默认下载位置。
    随后，转盘区域将被隐藏，界面进入只读状态。

---

### 界面与控制

- **背景调整控件**（位于左下角）

  - **纵向滑块**：控制背景模糊程度
  - **横向滑块**：调整白色覆盖层的不透明度
- **直播优化建议**
  在抽取过程中，可适当调整背景模糊度与白膜不透明度以增强视觉效果。
  抽取结束后，建议将两个滑块调回 **0**，恢复清晰背景。
  如有需要可以单独禁止浏览器页面的音频播放。

---

### 注意事项

❗ **重要操作提示**

- 工具界面为固定布局，推荐在 **1080P** 分辨率下全屏使用。
- 点击转盘后，请等待结果弹窗完全显示再进行下一次操作。

  - 过快点击可能导致重复抽取或触发浏览器默认行为，影响直播效果。
  - 此为已知设计限制，请遵循操作节奏。

---

### 其他

- 本工具专为"交错核心第一届好友杯"定制，**请勿随意传播**。
- 如有界面显示或操作问题，请确认屏幕分辨率与操作间隔是否符合上述要求。
